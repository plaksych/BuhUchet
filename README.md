# BuhUchet

Учебное бухгалтерское приложение на `C#`, `WPF` и `.NET 8`.

## Возможности

**Учёт и отчётность:**
* журнал хозяйственных операций (добавление, редактирование, удаление);
* справочник контрагентов;
* план счетов;
* формирование оборотно-сальдовой ведомости (ОСВ);
* экспорт и импорт отчёта ОСВ в `JSON`;
* автосохранение данных в `data.json`.

**Авторизация:**
* регистрация пользователей с проверкой уникальности логина;
* вход по логину и паролю;
* пароли хранятся в виде SHA-256 хэша в `users.json`.

## Требования

* ОС: Windows 10/11 (x64)
* [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

## Установка

Скачайте `BuhUchet_Setup.exe` из раздела [Releases](../../releases) и запустите установщик.
При отсутствии .NET 8 установщик предложит его скачать.

## Запуск из исходников

1. Откройте `BuhUchet.sln` в Visual Studio 2022+.
2. Восстановите NuGet-пакеты.
3. Запустите проект (`F5`).

## Файлы данных

* `data.json` — основные данные приложения (журнал, контрагенты, счета);
* `users.json` — учётные записи пользователей;
* файл отчёта ОСВ (при экспорте): любой выбранный пользователем `.json`.
