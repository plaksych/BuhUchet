# BuhUchet

Учебное бухгалтерское приложение на `C#`, `WPF` и `.NET 8`.

## Возможности

- журнал хозяйственных операций (добавление, редактирование, удаление);
- справочник контрагентов;
- план счетов;
- формирование оборотно-сальдовой ведомости (ОСВ);
- экспорт и импорт отчета ОСВ в `JSON`;
- автосохранение данных в `data.json`.

## Запуск

1. Откройте `BuhUchet.sln` в Visual Studio 2022+.
2. Восстановите NuGet-пакеты.
3. Запустите проект (`F5`).

## Файлы данных

- основной файл приложения: `data.json`;
- файл отчета ОСВ (при экспорте): любой выбранный пользователем `.json`.
